Download for Windows

To download and install the Matrise application, please follow these steps:

  1. Click the "Download Matrise" button below to download the application.
  2. Once the download is complete, locate the downloaded file (typically named Release.zip) in your download directory.
  3. Right-click on the Release.zip file and select "Extract All..." from the context menu.
  4. Choose a destination folder where you want to extract the files and click "Extract".
  5. After extraction, navigate to the folder where the files were extracted. Find and double-click the installer (usually named Setup.exe or similar) to start the installation process.
  6. Follow the on-screen instructions to complete the installation of Matrise.

Important: You may receive a warning from your operating system regarding the installation. This is a standard security measure. Please click "More Info" and then "Run Anyway" to proceed with the installation.
